🔗 Send to Google Form (optional record-keeping)

This sends the order data straight into a Google Form (which auto-fills a linked Google Sheet) — no Google account permission screens, no Apps Script deployment. One-way only: it sends data out, it does not read anything back.

How to set this up (one-time, ~5 min)
  1. Go to forms.google.com → create a new blank form.
  2. Add one "Short answer" question for each field you want to capture. Suggested fields: Doc No, Order No, Order Date, Customer Name, P.O. No, Status, Item Code, Item Name, Sales Qty, Stock Qty, Balance to Make, Total Production Days, Delivery Date. (You can add more from the full field list below.)
  3. Once all questions are added, click the ⋮ (three dots) menu top-right → "Get pre-filled link".
  4. This opens your form in fill-in mode. For each question, type the exact matching field key (shown below) as the answer — e.g. type customerName into the "Customer Name" question box. This is just a temporary marker so this app can match your questions automatically; it won't appear in real submissions.
  5. Click "Get Link" at the bottom, then copy that long link.
  6. Paste it in the box below and click "Detect Fields".
